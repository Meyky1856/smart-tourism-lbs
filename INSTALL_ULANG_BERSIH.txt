PENTING: Jangan extract zip ini menimpa folder lama.
Hapus/rename dulu folder smart-tourism-lbs yang lama, lalu extract zip ini ke folder baru.

Cara install frontend:
1. cd smart-tourism-lbs/frontend
2. rm -rf node_modules
3. npm ci --no-audit --no-fund
4. npm run dev

Cara install backend:
1. cd smart-tourism-lbs/backend
2. rm -rf node_modules
3. npm ci --no-audit --no-fund
4. npm run dev

Kalau sebelumnya npm install loading lama, penyebabnya kemungkinan package-lock lama masih tersisa atau zip lama belum lengkap.
Versi ini sudah memakai package-lock dengan registry resmi npmjs.org.
